【个人博客搭建】01 Win10 + Hexo + Github + 域名

以前搭建过一次个人网站，但电脑出问题后就直接荒废了，换到了博客园。不过博客园中间有段时间整改，而且定制性虽然已经很不错了但对主题的支持仍有点不够，索性再建一次。

其实最关键的是域名一直续费着但是没用。

Github建仓

Github可以帮助开发者托管项目代码，并用Gitpages发布。可以简单地把Github理解成云服务器，如果不用Github的话，本地的博客项目想要被外部访问就必须进行内网穿透，用云服务器发布进行则省去了这一麻烦事（当然用花生壳或者Sakura FRP做穿透也不算麻烦）。

注册了Github账号后，点击New repository建仓，这个仓库就是放置博客项目的地方。注意仓库名必须设置为：你的用户名.github.io。其他信息先不修改。
